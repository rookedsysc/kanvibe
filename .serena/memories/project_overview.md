KanVibe is a Next.js/React/TypeScript application for managing AI coding-agent tasks on a kanban board, with real-time board updates, terminal session integration, git worktree automation, and optional Electron desktop support. The repo includes Next app code under src/app, shared libs under src/lib, UI components under src/components, hooks under src/hooks, Electron-specific code under electron and src/desktop, and tests colocated under __tests__ plus Playwright e2e tests in tests/e2e.
